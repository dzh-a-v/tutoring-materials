Question 1

Venn Diagrams and Conditional Probability

We use probability rules to find unknown regions, test independence, and place a new event.

Q1 - The Problem

What Are We Given?

Tasks

  • Find P(B).
  • Decide whether A and B are independent.
  • Use P(C | B) = P(C) to find p and q.
  • Draw an event D that is mutually exclusive with A but overlaps B.

Q1 - Definitions

Key Terms

Independent events

A and B are independent if knowing one happened does not change the probability of the other.

P(A and B) = P(A)P(B)

Conditional probability

P(C | B) means "probability of C, given B". We restrict attention to the B region.

P(C | B) = P(C and B) / P(B)

Mutually exclusive

Two events cannot happen at the same time.

A and D mutually exclusive means P(A and D) = 0

Total probability

All separate regions in the sample space add to 1.

Q1 - Approach

General Strategy

  1. Use the fact that all regions in the Venn diagram add to 1 to find p + q.
  2. Add the regions inside B to find P(B).
  3. Use the independence test P(A and B) = P(A)P(B).
  4. Translate P(C | B) = P(C) into an equation involving p.
  5. Use p + q to finish.

Q1(a) and Q1(b)

Finding P(B) and Testing Independence

First find the unknown total:

0.04 + 0.06 + p + q + 0.3 + 0.4 = 1
p + q = 0.2

Event B contains p, q, and 0.3.

P(B) = p + q + 0.3 = 0.2 + 0.3 = 0.5

For independence:

P(A) = 0.04 + 0.06 + p + q = 0.3
P(A and B) = p + q = 0.2
P(A)P(B) = 0.3 x 0.5 = 0.15

Since 0.15 is not equal to 0.2, A and B are not independent.

Q1(c)

Using Conditional Probability

The statement P(C | B) = P(C) means that the probability of C is unchanged when we know B has happened.

P(C | B) = P(C and B) / P(B) = p / 0.5

The event C contains the regions 0.06 and p, so:

P(C) = p + 0.06

Set these equal:

p / 0.5 = p + 0.06
2p = p + 0.06
p = 0.06

Then use p + q = 0.2.

q = 0.2 - 0.06 = 0.14

Q1(d)

Where Can Event D Go?

Reasoning

  • D must not overlap A at all.
  • D must overlap B, because P(B and D) > 0.
  • A valid D can sit partly inside B-only and partly outside B, as long as it avoids A.

Q1 Answers

Final Answers and Notes

Answers

  • (a) P(B) = 0.5
  • (b) A and B are not independent.
  • (c) p = 0.06, q = 0.14
  • (d) D overlaps B but does not overlap A.

Useful note

When a Venn question gives unknown regions, first add all regions to 1. This often gives a useful relationship before any advanced formula is needed.

Question 2

Correlation and Regression

We measure the strength of a linear relationship and build a line to predict paper spending from number of employees.

Q2 - The Problem

What Are We Given?

A company records, for 12 offices:

  • x = number of employees
  • p = amount spent on paper in a year, measured in hundreds of dollars

sum x = 93, Sxx = 148.25, sum p = 273, sum p^2 = 6602.72, sum xp = 2347, n = 12

The variable p is in hundreds of dollars, so p = 10 means $1000.

Q2 - Definitions

The Statistics in the Question

TermMeaningFormula
SxxCorrected spread of x-valuessum x^2 - (sum x)^2 / n
SppCorrected spread of p-valuessum p^2 - (sum p)^2 / n
SxpCorrected joint variation between x and psum xp - (sum x)(sum p) / n
PMCC, rProduct moment correlation coefficientr = Sxp / sqrt(Sxx Spp)
Regression lineLine used to estimate p from xp = a + bx

Q2 - Approach

General Strategy

  1. Use the corrected product formula to show Sxp.
  2. Calculate Spp, because PMCC needs it.
  3. Use r = Sxp / sqrt(Sxx Spp).
  4. Find regression gradient b = Sxp / Sxx.
  5. Use means xbar and pbar to find intercept a.
  6. Apply the director's new model and compare with the original average spending.

Q2(a) and Q2(b)

Calculating Sxp and r

For corrected joint variation:

Sxp = sum xp - (sum x)(sum p) / n
Sxp = 2347 - (93 x 273) / 12
Sxp = 2347 - 2115.75 = 231.25

Now find Spp:

Spp = sum p^2 - (sum p)^2 / n
Spp = 6602.72 - 273^2 / 12 = 391.97

Then:

r = 231.25 / sqrt(148.25 x 391.97) = 0.959307...

PMCC: r = 0.959 to 3 significant figures.

Q2(c)

Regression Line of p on x

For the regression line p = a + bx, the gradient is:

b = Sxp / Sxx = 231.25 / 148.25 = 1.559865...

The line passes through the mean point (xbar, pbar).

xbar = 93 / 12 = 7.75
pbar = 273 / 12 = 22.75

Use a = pbar - b xbar.

a = 22.75 - 1.559865... x 7.75 = 10.661...

Regression line: p = 10.7 + 1.56x

Q2(d)

Interpreting the Gradient

The gradient tells us the change in predicted p when x increases by 1.

b = 1.56

Since p is measured in hundreds of dollars:

1.56 hundreds of dollars = 1.56 x 100 = 156 dollars

For each extra employee, the model predicts about $156 more spent on paper per year.

Q2(e)

Director's Saving Model

The director's model reduces the intercept to 4/5 a and the employee part to 1/2 bx.

Use the average number of employees across the 12 offices:

xbar = 93 / 12 = 7.75

New predicted average spending:

new p = (4/5)(10.661...) + (1/2)(1.559865...)(7.75)
new p = 14.573...

Original average spending was:

pbar = 273 / 12 = 22.75

Percentage saving:

((22.75 - 14.573...) / 22.75) x 100 = 35.94...

Estimated percentage saving: 36%

Q2 Answers

Final Answers and Notes

  • (a) Sxp = 231.25
  • (b) r = 0.959
  • (c) p = 10.7 + 1.56x
  • (d) About $156 extra per additional employee per year.
  • (e) Saving is about 36%.

Useful note

Always check units. In this question, the regression line predicts hundreds of dollars, not dollars.

Question 3

Stem-and-Leaf, Box Plots, IQR, and Outliers

We summarise passenger ages and use a box plot to compare distributions.

Q3 - The Problem

What Are We Given?

A stem-and-leaf diagram gives the ages of 35 male passengers.

StemLeaves
13
27, 9
31, 2, 8, 8
45, 5, 6, 7, 8, 8, 9
52, 2, 3, 3, 4, 4, 5, 6, 6, 8
60, 1, 1, 4, 4, 4, 7
73, 6
87, 8
Key: 1 | 3 means 13 years.

Q3 - Definitions

Summary Statistics

Median

The middle value after the data is in order.

position = (n + 1) / 2

Quartiles

Values that split ordered data into quarters. Q1 is the lower quartile and Q3 is the upper quartile.

IQR

Interquartile range. It measures the spread of the middle 50%.

IQR = Q3 - Q1

Outlier rule

Outliers are unusually far from the central data.

below Q1 - 1.5 IQR
above Q3 + 1.5 IQR

Q3 - Approach

General Strategy

  1. Use the ordered stem-and-leaf data to locate the median and quartiles.
  2. Calculate the IQR from Q3 - Q1.
  3. Calculate the lower and upper outlier limits.
  4. Draw the box plot using Q1, median, Q3, whiskers, and outliers.
  5. Compare male and female box plots using named statistics, not vague impressions.

Q3(a) and Q3(b)

Median and IQR

There are n = 35 values.

median position = (35 + 1) / 2 = 18th value

Counting through the ordered data, the 18th value is 53.

For the quartiles, the lower and upper middle positions give:

Q1 = 45
Q3 = 61

Therefore:

IQR = Q3 - Q1 = 61 - 45 = 16

Median = 53, and the IQR is 16.

Q3(c)

Showing There Are 3 Outliers

Use the outlier limits:

lower limit = Q1 - 1.5 x IQR = 45 - 1.5 x 16 = 21
upper limit = Q3 + 1.5 x IQR = 61 + 1.5 x 16 = 85

So outliers are:

values < 21 or values > 85

From the data, the values are:

13, 87, 88

There are 3 outliers.

Q3(d)

Drawing the Male Box Plot

Whiskers go to the smallest and largest non-outlier values: 27 and 76. Outliers are plotted separately.

Q3(e)

Comparing Male and Female Ages

To compare distributions, name a statistic and state its value.

Male passengers

Median age = 53

Female passengers

Median age = 67

A valid comment: the female passengers are generally older, because their median age is 67 compared with 53 for the male passengers.
The mark depends on using a statistic correctly. A sentence like "the female box plot is further right" is too vague on its own.

Q3(f)

Adding Anja's Family Without Changing the Female Box Plot

If adding 4 new ages does not change the female box plot, the new values must fit inside the existing sections of the distribution.

Granddaughter

She is younger than both daughters. To keep the box plot unchanged, she can be in the lower section but must not create a new minimum or move Q1.

34 <= granddaughter age <= 56

Anja

Anja is 23 years older than her eldest daughter. The eldest daughter can be in the upper middle section, about 67 to 72.

Anja = eldest daughter + 23
90 <= Anja age <= 93

Q3 Answers

Final Answers and Notes

  • (a) Median = 53
  • (b) Q1 = 45, Q3 = 61, IQR = 16
  • (c) Outliers: 13, 87, 88
  • (d) Box: 45 to 61, median 53; whiskers 27 and 76; outliers 13, 87, 88.
  • (e) Females generally older: median 67 > 53.
  • (f) Granddaughter: 34 to 56; Anja: 90 to 93.

Useful note

For box plots with outliers, whiskers stop at the last value that is not an outlier.

Question 4

Tree Diagrams and Discrete Distributions

We track a three-stage random process, then turn the outcomes into a probability distribution.

Q4 - The Problem

The Game

  • Bag A contains 4 red and 2 yellow balls.
  • Bag B contains 4 red and 1 yellow balls.
  • Bag C starts with 6 red balls.
  • Draw from A, then B, placing each drawn ball into C.
  • Finally draw from C.
The probabilities for the final draw from C depend on what happened in the first two draws.

Q4 - Definitions

Tree Diagram Language

Branch

One possible outcome at a stage of the experiment, with its probability written on it.

Path

A complete route through the tree, such as RYY.

Multiply along a path

The probability of a sequence is the product of branch probabilities.

Add separate paths

If an event can happen in several mutually exclusive ways, add those path probabilities.

Q4 - Approach

General Strategy

  1. Complete Bag A and Bag B probabilities first.
  2. For each A-B path, update Bag C before finding the final probabilities.
  3. For exactly 2 yellow balls, list the valid paths: RYY, YRY, YYR.
  4. Use conditional probability for part (c).
  5. For X, group paths by the number of yellow balls recorded.
  6. Use E(X) = sum x P(X = x).

Q4(a)

Completed Tree Diagram

For Bag C, update the contents after the first two draws: RR gives 8 red and 0 yellow; one yellow gives 7 red and 1 yellow; YY gives 6 red and 2 yellow.

Q4(b)

Exactly Two Yellow Balls

Exactly two yellows can happen in three ways:

RYY, YRY, YYR

Calculate each path:

P(RYY) = (2/3)(1/5)(1/8) = 2/120
P(YRY) = (1/3)(4/5)(1/8) = 4/120
P(YYR) = (1/3)(1/5)(6/8) = 6/120

Add them because the paths are separate:

2/120 + 4/120 + 6/120 = 12/120 = 1/10

Probability of exactly two yellow balls is 1/10.

Q4(c)

Conditional Probability from the Exactly-Two Event

We already know:

P(exactly two yellows) = 1/10 = 12/120

For the ball from Bag A to be red while exactly two yellows happen, the only path is RYY.

P(RYY) = 2/120

So:

P(A is red | exactly two yellows) = (2/120) / (12/120) = 1/6

Answer: 1/6

Q4(d)

Probability Distribution of X

X is the number of yellow balls recorded in the game. Possible values are 0, 1, 2, 3.

xPathsP(X = x)
0RRR64/120 = 8/15
1RYR, YRR42/120 = 7/20
2RYY, YRY, YYR12/120 = 1/10
3YYY2/120 = 1/60
Check: probabilities add to (64+42+12+2)/120 = 1.

Q4(e)

Expected Number of Yellow Balls

The expected value is the long-run average result if the game is repeated many times.

E(X) = sum x P(X = x)

Substitute the distribution:

E(X) = 0(64/120) + 1(42/120) + 2(12/120) + 3(2/120)
E(X) = (42 + 24 + 6) / 120 = 72/120 = 0.6

Expected number of yellow balls: E(X) = 0.6

Q4 Answers

Final Answers and Notes

  • (a) Tree probabilities as listed.
  • (b) P(exactly 2 yellows) = 1/10
  • (c) P(A red | exactly 2 yellows) = 1/6
  • (d) Distribution: 8/15, 7/20, 1/10, 1/60 for x=0,1,2,3.
  • (e) E(X)=0.6

Useful note

When a later bag changes because of earlier draws, the final branch probabilities must be different on different paths.

Question 5

Discrete Random Variables

We use symmetry, cumulative probability, variance, and a transformation from Y to a distance D.

Q5 - The Problem

Probability Distribution of Y

y-9-5059
P(Y = y)qrurq
F(0) = 19/30 Var(Y) = 37 P = (12, Y) D = OP

We need E(Y), u, q, r, and the probability distribution of D.

Q5 - Definitions

Random Variable Tools

Discrete random variable

A variable that can take separate values, each with a probability.

Expected value

The long-run average.

E(Y) = sum y P(Y = y)

Variance

A measure of spread around the mean.

Var(Y) = E(Y^2) - [E(Y)]^2

Cumulative distribution function

F(y) = P(Y <= y). So F(0) adds probabilities at -9, -5, and 0.

Q5 - Approach

General Strategy

  1. Use symmetry to write down E(Y).
  2. Use F(0)=19/30 and total probability = 1 to solve for u.
  3. Use variance to create a second equation for q and r.
  4. Use D = sqrt(12^2 + Y^2) to convert each possible Y value into a distance.
  5. Add probabilities for Y values that give the same distance.

Q5(a)

Expected Value by Symmetry

The distribution is symmetric about 0:

PairProbabilitiesEffect on mean
-9 and 9both qcancel
-5 and 5both rcancel
0ucontributes 0

E(Y) = -9q - 5r + 0u + 5r + 9q = 0

Answer: E(Y) = 0

Q5(b)

Showing u = 4/15

From the cumulative distribution function:

F(0) = P(Y <= 0) = q + r + u = 19/30

Total probability is 1:

2q + 2r + u = 1

Rewrite the second equation as:

2(q + r) + u = 1

Since q + r + u = 19/30, subtract to get:

q + r = 1 - 19/30 = 11/30

Then:

u = 19/30 - 11/30 = 8/30 = 4/15

Q5(c)

Finding q and r from the Variance

Because E(Y)=0, the variance is just E(Y^2).

E(Y^2) = (-9)^2q + (-5)^2r + 0^2u + 5^2r + 9^2q
E(Y^2) = 162q + 50r

Given Var(Y)=37:

162q + 50r = 37

Also from part (b):

q + r = 11/30

Solve by substituting r = 11/30 - q:

162q + 50(11/30 - q) = 37
112q + 55/3 = 37
q = 1/6

r = 11/30 - 1/6 = 1/5

Q5(d)

Turning Y into the Distance D

The point is P = (12, Y). The distance from the origin is found using Pythagoras.

D = sqrt(12^2 + Y^2)

Use the values already found: u = 4/15, r = 1/5, and q = 1/6.

Y valueD valueProbability contribution
0sqrt(144) = 12u = 4/15
-5 or 5sqrt(144 + 25) = 13r + r = 2r = 2/5 = 6/15
-9 or 9sqrt(144 + 81) = 15q + q = 2q = 1/3 = 5/15
The key idea is that opposite Y-values give the same distance because the formula uses Y^2.

Q5(d)

Distribution of D

d121315
P(D = d)4/156/15 or 2/55/15 or 1/3

P(D = 12) = P(Y = 0) = 4/15
P(D = 13) = P(Y = -5 or Y = 5) = 1/5 + 1/5 = 2/5
P(D = 15) = P(Y = -9 or Y = 9) = 1/6 + 1/6 = 1/3

The probabilities add to 4/15 + 6/15 + 5/15 = 1, or equivalently 4/15 + 2/5 + 1/3 = 1.

Q5 Answers

Final Answers and Notes

  • (a) E(Y)=0
  • (b) u=4/15
  • (c) q=1/6, r=1/5
  • (d) D=12,13,15 with probabilities 4/15, 6/15 or 2/5, 5/15 or 1/3.

Useful note

When a transformation uses Y^2, opposite values such as -5 and 5 often merge into the same new value.

Question 6

Normal Distribution and Shelf Design

We convert book heights to z-scores, use percentiles, then translate the result into a design height.

Q6 - The Problem

Book Heights

The height H of books is modelled by a normal distribution:

H ~ N(25.1, 5.5^2)

Mean

25.1 cm. The centre of the normal model.

Standard deviation

5.5 cm. Typical spread away from the mean.

Xiang excludes the smallest 5% of books and all books taller than 30.8 cm.

Q6 - Definitions

Normal Distribution Tools

Normal distribution

A bell-shaped continuous model, symmetric about the mean.

z-score

The number of standard deviations a value is from the mean.

z = (x - mean) / sd

Percentile

A cutoff with a given percentage below it. The 5th percentile has 5% below.

Tail probability

A probability in one end of the normal curve, such as P(H > 30.8).

Q6 - Approach

General Strategy

  1. Standardise heights using z = (x - 25.1)/5.5.
  2. Use normal tables or calculator values for probabilities.
  3. Use inverse normal values for percentile cutoffs.
  4. Translate ratios 2 : 3 : 3 into proportions of the placed books.
  5. Convert the shelf categories into physical shelf heights.

Q6(a)

Showing P(H > 30.8) = 0.15

Standardise 30.8 cm:

z = (30.8 - 25.1) / 5.5 = 1.03636...

So:

P(H > 30.8) = P(Z > 1.03636...)

Using tables, Phi(1.04) = 0.8508. This is the probability to the left.

P(Z > 1.04) = 1 - 0.8508 = 0.1492

Therefore P(H > 30.8) = 0.15 to 2 significant figures.

Q6(b)

Range of Heights Placed on Shelves

Books taller than 30.8 cm are excluded. The smallest 5% are also excluded, so find the 5th percentile.

For the standard normal distribution:

P(Z < -1.6449) = 0.05

Set up the standardisation equation:

(y - 25.1) / 5.5 = -1.6449

Solve:

y = 25.1 + 5.5(-1.6449) = 16.053...

Placed book heights are approximately 16.1 cm to 30.8 cm.

Q6(c)(i)

Showing d = 25.8

Placed books make up 80% of all books, because 5% are too small and 15% are too tall.

The ratio small : medium : large is 2 : 3 : 3, total 8 parts.

small = 2/8 of placed = 0.20 of all books
medium = 3/8 of placed = 0.30 of all books
large = 3/8 of placed = 0.30 of all books

The upper boundary d is after the bottom 5%, small 20%, and medium 30%:

P(H < d) = 0.05 + 0.20 + 0.30 = 0.55

The z-value for 0.55 is about 0.126.

d = 25.1 + 5.5(0.126) = 25.79...

So d = 25.8 cm to 1 decimal place.

Q6(c)(ii)

Finding m

The boundary m is between small and medium books.

It comes after the bottom excluded 5% and the small placed 20%:

P(H < m) = 0.05 + 0.20 = 0.25

The z-value for 0.25 is about -0.67449.

(m - 25.1) / 5.5 = -0.67449

Solve:

m = 25.1 + 5.5(-0.67449) = 21.390...

Answer: m = 21.4 cm to 3 significant figures.

Q6 Visual

How the Normal Curve Is Split

The diagram is a concept sketch: it shows the probability regions, not exact scale.

Q6(d)

Minimum Total Height for Shelves

The shelves are planned as:

  • 2 shelves for small books, each needs height m = 21.4
  • 3 shelves for medium books, each needs height d = 25.8
  • 3 shelves for large books, each needs height 30.8
  • Wood thickness: 1 cm per shelf layer, total 8 cm in the mark scheme model.

height = 2(21.4) + 3(25.8) + 3(30.8) + 8
height = 42.8 + 77.4 + 92.4 + 8 = 220.6

Minimum total height needed: 221 cm to the nearest cm.

Q6 Answers

Final Answers and Notes

  • (a) P(H > 30.8) = 0.1492... = 0.15
  • (b) Range: about 16.1 cm to 30.8 cm
  • (c)(i) d = 25.8 cm
  • (c)(ii) m = 21.4 cm
  • (d) Minimum total height: 221 cm

Useful note

Ratios in a restricted group must be converted into probabilities for the whole distribution before using normal tables.

Exam Technique

Common Mistakes in This Paper

  • Testing independence using P(A) = P(B) instead of P(A and B) = P(A)P(B).
  • Forgetting that p in Question 2 is measured in hundreds of dollars.
  • Using outliers as whisker endpoints in a box plot.
  • Adding tree path probabilities before multiplying branches.
  • Forgetting to combine probabilities when different Y values give the same D value.
  • Using 0.80 directly as a cumulative probability for a normal boundary in Question 6.
